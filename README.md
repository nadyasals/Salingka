# Salingka — Into the Wild Heart of Sumatra

Salingka is a premium, cinematic ecotourism landing page built for the **SDGs Creative Web Competition (BYTESFEST 2026)**. It targets **Sub-theme 3 (Eco Journey: Sustainable Tourism Experience)**, highlighting sustainable tourism in Sumatra in alignment with **SDG 8** (Decent Work and Economic Growth) and **SDG 15** (Life on Land).

The site features a dark, immersive design using **Liquid Glass** morphism UI styling, staggered animations, a custom 3D Orbital Explorer navigation map, and sand disintegration transitions.

---

## 🚀 Getting Started

This project is built using **Next.js**, **React**, **TypeScript**, and **Tailwind CSS**. Follow these steps to set up and run the project locally.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed (v18.x or later recommended).

### 📦 Installation

1. Open a terminal in the root of the project directory.
2. Install the dependencies using `pnpm` (recommended) or `npm`:
   ```bash
   npm install
   # or
   pnpm install
   ```

### 💻 Running the Development Server

Start the local Next.js dev server:
```bash
npm run dev
# or
   pnpm dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser to view the site.

### 🏗️ Building for Production

Compile and optimize the site for production build:
```bash
npm run build
# or
pnpm build
```
To run the production build locally:
```bash
npm run start
# or
pnpm start
```

## 🌿 SDGs & Subtheme Features

- **SDG 15 (Life on Land)**: Explores Sumatran wildlife protection (Critically Endangered species counts) and rainforest conservation (2.6M hectares of Leuser UNESCO Biosphere).
- **SDG 8 (Decent Work & Economic Growth)**: Focuses on local community empowerment, showing how tourism revenue supports local guides, homestays, and family livelihoods.
- **Interactivities**: An interactive 3D Orbital Map for ecosystem navigation, sand disintegration canvas effects, and responsive liquid-glass CTA cards.
- **Atmospheric Effects**: Realistic floating leaves, fireflies, and fog elements that drift across the page as the user scrolls.
